# reviews-API

API:
| url               | parameters            |
| -------------     |:-------------:        |
| /auth/register    | username, password    |
| /artists/         | Optional: name        |
| /albums/          | Optional: name        |
| /latest/          |                       |
| /reviewer/:name   | name                  |
| /reviews/between/ | start, end            |
| /save-review/     | text, album, artist, token       |
